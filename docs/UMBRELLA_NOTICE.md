# 우산 부착용 유의사항

학생이 바로 읽고 따라할 수 있게 **짧고 명확하게** 만든 인쇄용 안내문입니다. 우산 손잡이 근처나 QR 코드 옆에 함께 붙이세요.

> **인쇄 팁**
> - 아래 "A. 스티커 버전" 은 6 × 9 cm 정도 작은 스티커지에 출력
> - 아래 "B. 게시판용 포스터" 는 A4 1장에 출력해 우산 보관대 근처에 부착
> - 이모지가 인쇄에서 흐리게 나오면 기호(●, ◆, ☂) 로 대체

---

## A. 스티커 버전 (우산마다 1장씩 부착)

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

     ☂ 공유 우산 001번

     [빌릴 때]
     QR 을 스캔하고 학번·이름·전화번호를 입력

     [돌려줄 때]
     같은 QR 을 다시 스캔
     → 거치대에 건 우산을 사진으로 촬영

     ┏━━━━━━━━━━━━━━━━━━━━━━━━┓
     ┃   반납은 다음날(익일) 까지    ┃
     ┃   주말·공휴일이면 첫 평일에  ┃
     ┗━━━━━━━━━━━━━━━━━━━━━━━━┛

     파손·분실 시 담당자에게 즉시 알려주세요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

> 위 "001번" 부분은 **우산 번호에 맞게 출력 시 교체**하세요.

## B. 게시판용 포스터 (보관대 근처 1장)

```
═══════════════════════════════════════════════

            ☂  공 유  우 산  사 용 법  ☂

═══════════════════════════════════════════════

  1. 빌리기
  ─────────────────────────────────────────────
  ① 우산의 QR 코드를 스마트폰 카메라로 스캔
  ② 학번 · 이름 · 전화번호 입력
  ③ "우산 빌리기" 버튼 → 완료

  2. 돌려주기
  ─────────────────────────────────────────────
  ① 우산을 보관대(거치대)에 제대로 걸기
  ② 같은 QR 을 다시 스캔
  ③ 학번 · 이름 입력
  ④ 우산이 거치대에 걸린 모습을 사진 촬영
  ⑤ "반납 완료하기" 버튼 → 완료

  3. 반납 기한
  ─────────────────────────────────────────────
  ◆ 다음날(익일) 까지 반납이 원칙입니다.
  ◆ 다음날이 주말·공휴일이면 가장 빠른 평일에
     반납해주세요.
     예) 금요일 대여 → 월요일 반납
         명절 연휴 직전 대여 → 연휴 다음 평일 반납

  4. 유의사항
  ─────────────────────────────────────────────
  ● 대여·반납 모두 QR 스캔이 필수입니다.
    종이 명부 없이 전부 전자 기록으로 관리됩니다.
  ● 반납 사진은 우산과 거치대가 함께 보이도록
    찍어주세요. (우산만 찍으면 반납 무효)
  ● 파손 · 분실 시 즉시 담당자에게 알려주세요.
    무단 반납이 확인되면 변상 책임이 따를 수
    있습니다.
  ● QR 이 훼손되어 인식이 안 되면 담당자에게
    교체를 요청해주세요.

  5. 문의
  ─────────────────────────────────────────────
  담당자:  _____________________
  연락처:  _____________________

═══════════════════════════════════════════════
```

## C. 초간단 스티커 (QR 바로 옆 작은 문구)

QR 코드 바로 아래 붙일 **한 줄짜리 안내문** 입니다. QR 스티커에 텍스트 공간이 부족할 때 별도로 덧붙이세요.

```
스캔 → 빌리기 / 다시 스캔 → 반납 · 다음 평일까지
```

또는

```
빌릴 때 / 돌려줄 때 모두 이 QR 을 스캔하세요
반납 기한: 다음 평일
```

## D. 담당자 참고사항

### 인쇄물 디자인 팁
- 비 맞는 환경이므로 **방수 스티커지** 또는 투명 테이프로 코팅
- 우산 색상이 다양하면 **흰 배경 + 검정 글씨** 가 가장 잘 보임
- QR 코드와 안내문이 **같은 방향으로** 보이도록 부착

### 문구 수정이 필요한 경우
- 반납 정책이 바뀌면 **3번 "반납 기한"** 부분만 수정
- 담당자 바뀌면 **5번 문의** 의 연락처만 수정
- 기타 운영 방침 변경 시 이 파일을 업데이트 후 재인쇄

### 시즌별 권장 공지
- **비 많은 계절 (6~9월)**: "반납 지연 시 다른 학생이 빌릴 수 없으니 기한 엄수" 강조 문구 추가
- **방학 직전**: "방학 전 전원 반납 요청" 공지 별도 부착
